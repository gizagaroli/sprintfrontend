# CarePlus

Aplicativo de saúde gamificado, com foco no acompanhamento de hábitos saudáveis, desafios e interação entre usuários.

## Objetivo

O projeto tem como objetivo incentivar hábitos saudáveis por meio de gamificação, recompensas e interação social entre colaboradores. A proposta incial é que seja um aplicativo mobile.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap
- Font Awesome

## Funcionalidades

- Página inicial com acesso rápido
- Sistema de desafios gamificados
- Acúmulo de moedas 
- Marketplace de recompensas
- Perfil do usuário com métricas de saúde
